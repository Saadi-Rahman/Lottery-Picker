  1. Enter a participant name in the input field.
  2. Click the Add button to add the participant.
  3. Repeat until all participants have been added.
  4. Choose one of the following draw methods: Random Pick or Quick Draw.
  5. Use Random Pick to select participants one at a time in random order.
  6. Each selected participant is removed from the participant list and added to the results list.
  7. Continue using Random Pick until all participants have been selected.
  8. Or use Quick Draw to instantly generate the complete randomized result order in a single click.
  9. When all participants have been selected, the lottery ends automatically.
  10. Click the refresh button to start a new lottery.
