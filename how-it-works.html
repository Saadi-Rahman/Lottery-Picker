📖 How It Works

  1. Enter a participant name in the input field.
  2. Click the OK button to add the participant.
  3. Repeat until all participants have been added.
  4. Click Random Pick to select a participant randomly.
  5. The selected participant is removed from the participant list.
  6. The selected participant is added to the results list.
  7. Continue clicking Random Pick until all names are selected.
  8. When no participants remain, the lottery ends automatically.
  9. Use the refresh button to start a new lottery.
